# TravelBharat 🇮🇳

**TravelBharat** is a modern, full-stack tourism management platform built using the **MERN stack**. It is designed to simplify travel within India by providing a centralized hub where users can explore states, discover tourist attractions, mark favorites, and book hotel packages.

## ✨ Features

### 👤 User Features
* **Secure Authentication:** JWT-based login and registration with password hashing using BcryptJS.
* **State & Place Exploration:** Browse detailed information about various Indian states and specific tourist spots.
* **Personalized Favorites:** Logged-in users can save their favorite locations to a personal list.
* **Booking System:** Seamlessly book hotels and travel packages.
* **User Reviews:** Provide and read feedback for specific tourist destinations.
* **My Trips:** A dedicated section to track and manage personal travel bookings.

### 🛠 Admin Features
* **Content Management:** Full CRUD (Create, Read, Update, Delete) operations for states, places, and packages.
* **Booking Management:** Oversee all user bookings and system activities.

## 🚀 Tech Stack

**Frontend:**
* React.js (Vite)
* Tailwind CSS (Styling)
* React Router (Navigation)
* Axios (API Integration)

**Backend:**
* Node.js & Express.js
* MongoDB (NoSQL Database)
* Mongoose (ODM)
* JWT (Authentication)
